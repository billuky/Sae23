Présentation de la SAé Numéro 23

Bienvenue sur le site réalisé dans le cadre de la SAÉ23 du BUT Réseaux & Télécommunications. Ce projet a pour objectif de mettre en place une solution complète permettant la récupération, le traitement et l'affichage de données issues de capteurs répartis dans les bâtiments de l’IUT.

Objectifs du projet

  • Collecter automatiquement les données (ex. luminosité) depuis un broker MQTT
  • Enregistrer les mesures dans une base de données MySQL
  • Afficher les statistiques (min, max, moyenne) dans une interface web
  • Permettre à un gestionnaire de consulter uniquement les capteurs de son bâtiment
  • Automatiser le tout via des scripts Bash et un planificateur de tâches (crontab)
