Livrable 2

Flow Node-Red

Pour ce livrable nous avons déployé des nœuds connectés ensembles et configurés sur les éléments essentiels afin d’afficher les données récupérées sur une interface graphique.

Ce flow Node-RED est conçu pour collecter, stocker et visualiser des données de tous les types provenant tous les capteurs distincts. Chaque capteur envoie ses mesures via MQTT, et ces messages sont directement récupérés par Node-RED. Une fois reçues, les données sont simultanément enregistrées dans une base InfluxDB (influxdbRT) pour permettre un archivage historique et une analyse ultérieure.

Ainsi, par une interface que uniquement l'administrateur ne peut accéder, on peut y voir toutes les données de tous les capteurs. Comme affiché ci-dessous pour la salle E208 :
